# Healthcare Appointment Queue System

## Overview

The Healthcare Appointment Queue System is a full-stack web application designed to manage hospital appointments and patient queues efficiently. It allows patients to book appointments, while doctors and administrators can manage schedules and monitor queue flow.

---

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA
* REST APIs

### Frontend

* React.js
* HTML, CSS, JavaScript

### Database

* H2 Database (In-Memory)

---

## Features

* Patient registration and management
* Doctor management
* Appointment booking system
* Queue management and tracking
* Dashboard for viewing statistics (patients, doctors, appointments)
* Integration of frontend and backend using REST APIs

## How to Run the Project

### Backend

1. Open the backend project in Spring Tool Suite / IntelliJ / VS Code
2. Run the main Spring Boot application
3. The server will start at:
   http://localhost:8080

---

### Frontend

1. Navigate to the frontend folder:
   cd Healthcarefrontend

2. Install dependencies:
   npm install

3. Run the application:
   npm start

4. Open in browser:
   http://localhost:3000
